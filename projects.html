Covid-19 Info Tracker
Covid-19 Info Tracker is an Android application built using Java and XML. It is an India specific tracker which allows user to track updated and valid data(related to Covid-19) of various Indian regions.

The application allows user to compare data of 2 Indian regions based on parameters and visulaize it in a graphical format.It allows a user to find vaccine centres nearby.It also helps a user to contact official govt. helpdesks and websites.
